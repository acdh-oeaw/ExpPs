Accessing the Edition - Canonical Text Service

Requests

Endpoint

The address for making CTS queries is:


https://expps.acdh-dev.oeaw.ac.at/psalmcatenae-server/cts/

Structure of the URNs

An identifier of an fragment of the edition looks like this:


urn:cts:etf:psath.expps.{language}:{number of passage}

The value for language can be grc for the greek text or ger for the german translation.

The value for number of passage must be the running number of the fragments of the edition. The identifier can also be found in the online version of the edition at the end of each commentary fragment. A list of all URNs is delivered for the request GetValidReff (see below).

Exemplaric Request


https://expps.acdh-dev.oeaw.ac.at/psalmcatenae-server/cts/?request=GetPassage&urn=urn:cts:etf:psath.expps.grc:1

This request delivers as result the first fragment of the edition in greek language.

Response

The result for the exemplaric request above is:


<GetPassage>
   <request>
      <requestName>GetPassage</requestName>
      <requestUrn>urn:cts:etf:psath.expps.grc:1</requestUrn>
   </request>
   <reply>
      <urn>urn:cts:etf:psath.expps.grc:1</urn>
      <passage>
         Τὴν ἀρχὴν τῆς προφητείας ... καὶ λοιμοί: –
      </passage>
      <license>
         Available under the Creative Commons Attribution 4.0 International (CC BY 4.0)
      </license>
      <source>
         Heil, Uta; Panteghini, Sebastiano, (Ps.)-Athanasius, Expositiones in Psalmos. Fragment 1 - urn:cts:etf:psath.expps.grc:1 (https://expps.acdh-dev.oeaw.ac.at/expps/edition.html).
      </source>
   </reply>
</GetPassage>

Further Requests

GetValidReff

To get a list of all available URNs the following request can be used.


https://expps.acdh-dev.oeaw.ac.at/psalmcatenae-server/cts/?request=GetValidReff

GetLabel

To get a short label for a particular commentary fragment the following request can be used.


https://expps.acdh-dev.oeaw.ac.at/psalmcatenae-server/cts/?request=GetLabel&urn=urn:cts:etf:psath.expps.grc:1

Accessing the Transcriptions - REST API

The REST API has the following base URL:


https://expps.acdh-dev.oeaw.ac.at/psalmcatenae-server/

All URL fragments of the endpoints are attached to this base URL

Manuscripts

manuscripts

Returns a list of all manuscripts contained in the database.

Commentary Fragments

commentaryfragments

Returns a list of all commentary fragments of all manuscripts contained in the database.

manuscripts/{manuscript-id}/commentaryfragments

Returns a list of all commentary fragments of a particular manuscript.

manuscripts/{manuscript-id}/commentaryfragments/{commentaryfragment-id}

Returns a particular commentary fragment of a particular manuscript.

manuscripts/{manuscript-id}/commentaryfragments/search

Parameters:

  1. author - The author of a fragment according to the manuscript.
  2. author-critical - The author critically attributed of a commentary fragment.
  3. reference - The commented psalmverse.
  4. reference-from - The start of a range of commented psalmverses.
  5. reference-to - The end of a range of commented psalmverses.

With this endpoint and the parameters it is possible to search for fragments of a particular author, for commentary fragments concerning a particular psalmverse, or for fragments commenting on a range of psalmverses.

Examples:


https://expps.acdh-dev.oeaw.ac.at/psalmcatenae-server/manuscripts/vat-gr-754/commentaryfragments/search?author=Athanasius
				

https://expps.acdh-dev.oeaw.ac.at/psalmcatenae-server/manuscripts/vat-gr-754/commentaryfragments/search?reference=Ps (LXX) 7,10
				

https://expps.acdh-dev.oeaw.ac.at/psalmcatenae-server/manuscripts/vat-gr-754/commentaryfragments/search?reference-from=Ps (LXX) 9,1&reference-to=Ps (LXX) 9,4

Hexaplaric Variants

hexaplaric-variants

Returns a list of all hexaplaric variants of all manuscripts contained in the database.

manuscripts/{manuscript-id}/hexaplaric-variants

Returns a list of all hexaplaric variants of a particular manuscript.

manuscripts/{manuscript-id}/hexaplaric-variants/{hexaplaric-variant-id}

Returns a particular hexaplaric variant of a particular manuscript.

manuscripts/{manuscript-id}/hexaplaric-variants/search

Parameter:

  1. reference - The psalmverse to which hexaplaric variants are connected.
  2. reference-from - The start of the range to which hexaplaric variants are connected.
  3. reference-to - The end of the range.

With this endpoint and the parameters it is possible to get hexaplaric variants connected to a psalmverse or a range of psalmverses.

Example:


https://expps.acdh-dev.oeaw.ac.at/psalmcatenae-server/manuscripts/vat-gr-754/hexaplaric-variants/search?reference-from=Ps (LXX) 9,1&reference-to=Ps (LXX) 9,4

Glosses

glosses

Returns a list of all glosses of all manuscripts contained in the database.

manuscripts/{manuscript-id}/glosses

Returns a list of all glosses of a particular manuscript.

manuscripts/{manuscript-id}/glosses/{gloss-id}

Returns a particular gloss of a particular manuscript.

manuscripts/{manuscript-id}/glosses/search

Parameter:

  1. author - The author of a gloss according to the manuscript.
  2. author-critical - The author critically attributed of a gloss.
  3. reference - The commented psalmverse.
  4. reference-from - The start of a range of commented psalmverses.
  5. reference-to - The end of a range of commented psalmverses.

With this endpoint and the parameters it is possible to search for glosses by an author or for glosses connected to a psalmverse or a range of psalmverses.

Example:


https://expps.acdh-dev.oeaw.ac.at/psalmcatenae-server/manuscripts/vat-gr-754/glosses/search?reference-from=Ps (LXX) 1,1&reference-to=Ps (LXX) 9,4

Authors

manuscripts/{manuscript-name}/authors

Returns a list of authors of fragments in a given manuscript.

manuscripts/{manuscript-name}/authors-distinct

Returns a list of distinct authors of fragments in a given manuscript.

manuscripts/{manuscript-name}/commentaryfragments/authors-distinct

Returns a list of distinct authors of commentary fragments in a given manuscript.

manuscripts/{manuscript-name}/glosses/authors-distinct

Returns a list of distinct authors of glosses in a given manuscript.

Authors Critical

manuscripts/{manuscript-name}/authors-critical

Returns a list of critically assigned authors of fragments in a given manuscript.

manuscripts/{manuscript-name}/authors-critical-distinct

Returns a list of distinct critically assigned authors of fragments in a given manuscript.

manuscripts/{manuscript-name}/authors-critical-normalized

Returns a list of normalized critically assigned authors of fragments in a given manuscript.

manuscripts/{manuscript-name}/commentaryfragments/authors-critical-normalized

Returns a list of normalized critically assigned authors of commentary fragments in a given manuscript.

manuscripts/{manuscript-name}/glosses/authors-critical-normalized

Returns a list of normalized critically assigned authors of glosses in a given manuscript.

References

manuscripts/{manuscript-name}/references

Returns a list of psalmverses for which fragments are available in a given manuscript.

manuscripts/{manuscript-name}/commentaryfragments/references

Returns a list of psalmverses for which commentary fragments are available in a given manuscript.

manuscripts/{manuscript-name}/hexaplaric-variants/references

Returns a list of psalmverses for which hexaplaric variants are available in a given manuscript.

manuscripts/{manuscript-name}/glosses/references

Returns a list of psalmverses for which glosses are available in a given manuscript.
